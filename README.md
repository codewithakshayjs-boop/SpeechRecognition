# 🎙️ Voice Assistant

A Python-based Voice Assistant that listens to your voice commands and performs tasks such as opening websites, telling the current time, recognizing speech, and responding with voice feedback. This project demonstrates the integration of Speech Recognition and Text-to-Speech technologies to create a simple hands-free virtual assistant.

## ✨ Features

* 🎤 Speech-to-Text Conversion
* 🔊 Text-to-Speech Responses
* 🌐 Open Websites Using Voice Commands
* 🕒 Get Current Date and Time
* 🤖 Execute Voice-Based Commands
* ⚡ Fast and Easy-to-Use Interface

## 🛠️ Technologies Used

* Python 3.x
* SpeechRecognition
* PyAudio
* pyttsx3
* webbrowser
* datetime

## 📦 Installation

```bash
git clone https://github.com/your-username/voice-assistant.git

```

## ▶️ Usage

Run the application:

```bash
python main.py
```

Example Commands:

* "Open Google"
* "Open YouTube"
* "What is the time?"
* "Tell me the current time"

## 🚀 How It Works

1. Listens to the user's voice input.
2. Converts speech into text.
3. Processes the recognized command.
4. Executes the requested action.
5. Responds using text-to-speech.
